# DemirTV

Premium aile odakli canli yayin uygulamasi (Android TV + telefon + tablet).

## Ozellikler
- Profil secimi
- Canli TV oynatma (HLS)
- Premium arayuz ve akici deneyim
- Guncellenebilir kanal listesi (M3U)

## Kurulum
1. Android Studio ile projeyi acin.
2. `app` modulunu calistirin.
3. Cihazda veya emulator/TV'de test edin.

## Kanal Listesi (M3U)
Bu uygulama yalnizca lisansli ve kullanim izni alinan yayinlar ile calistirilmalidir. M3U listesini yayin hakki size ait veya izinli kaynaklardan saglayin.

## Guncelleme
Uygulama, yeni surum kontrolu icin bir JSON guncelleme kaynagi kullanir. (Detaylar `update.json` ile ayarlanabilir.)
